# phone-number-validate

## Usage
```
phone-number-validate --help
Validates a list of phone numbers.

Usage:
  phone-number-validate [flags] phonenumber[...phonenumber]

Flags:
      --cfg string    config file containing a list of phone numbers
  -h, --help          help for phone-number-validate
      --onlyinvalid   Toggle only show invalid numbers
      --short         Toggle Short Format.
  -v, --version       version for phone-number-validate
```
